PRINT INFORMATION


ACCESS STUDENT'S PRINT ACCOUNT

 * Go to smartprint.cod.edu [https://smartprint.cod.edu]
 * Enter myACCESS login and password



ACCESS COMMUNITY USER'S PRINT ACCOUNT

If you are not a COD student, you will need to create a Guest Account by doing
the following:

 * Go to smartprint.cod.edu [https://smartprint.cod.edu]
 * Click “I am a guest” in the lower left-hand corner of the login box
 * Enter the requested information and click “Create”



ADD FUNDS TO PRINT ACCOUNT

When logged in to Print Account, click on "Add Funds" near the lower, center of
the screen. Use PayPal, credit or debit card to add funds online; minimum is $5.

To add funds with cash, go to the kiosk in the College of DuPage Library on the
Main Campus; minimum is $1 (no coins).

NOTE: Unused funds are not refundable


COST TO PRINT


BLACK AND WHITE PRINTS

 * 8 1/2" X 11" 1-sided: .06
 * 8 1/2" X 11" 2-sided: .11
 * 8 1/2" X 14" 1-sided: .06
 * 8 1/2" X 14" 2-sided: .11
 * 11" X 17" 1-sided: .12
 * 11" X 17" 2-sided: .22
   


COLOR PRINTS

 * 8 1/2" X 11" 1-sided: .10
 * 8 1/2" X 11" 2-sided: .19
 * 8 1/2" X 14" 1-sided: .10
 * 8 1/2" X 14" 2-sided: .19
 * 11" X 17" 1-sided: .20
 * 11" X 17" 2-sided: .38


PRINTING PROCEDURE

 * Print from laptop, phone, or tablet by clicking on "Upload" when logged in to
   Print Account, then browse file and upload to print queue.
 * Print from campus lab computers by choosing the print command from the
   application. Enter myACCESS login and password when prompted to upload to
   print queue.
 * Go to a Print Release Station at one of the Computer Lab Locations listed
   above and enter myACCESS login and password to display print queue; select
   print job for printing.  Appropriate funds will be deducted from the account
   balance.  NOTE:  All printers are set to print black and white two-sided by
   default.
 * Click on "Logoff" at end of printing session.


MOBILE PRINT

 * Using the same email account set up with printing system, send an email, with
   the documents you want printed as an attachment to: PrintBW@dupage.edu
   [PrintBW@dupage.edu] for Black and White print job and to 
   PrintColor@dupage.edu [PrintColor@dupage.edu] for Color print job
 * Wait for return email showing a list of documents submitted and instructions
   on how to release them.


IMPORTANT NOTE

 * Documents will be held in the print queue for 24 hours
 * For assistance with your Print Account, call the Student Support Help Desk at
   (630) 942-2999 or email at studenthelp@dupage.edu [studenthelp@dupage.edu].
