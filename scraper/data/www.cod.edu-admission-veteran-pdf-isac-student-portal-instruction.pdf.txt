ISAC Student Portal 
Program Applications and Status Check  
 
The Illinois Student Assistance Commission - ISAC Student Portal allows students to 
check the application status, view current benefit eligibility, and print an eligibility letter. 
 
1.  Go to ISAC.org 
2. Select ISAC Student Portal 
         
 
 
3.  Select Program Applications and Status Checks 
                  
 
 
 
 
 
 

4. Select the appropriate benefit status check: IVG, ING or MIA/POW Scholarship.     
                             
5. Log In or Sign Up
